# Локальный Backtrader

Это отдельный расчётный движок. Он запускается на этом компьютере, не связан с Telegram и не использует ИИ или лимиты Codex.

Сейчас в папке есть проверочная стратегия пересечения SMA 20/50. Она нужна, чтобы убедиться, что движок и исторические данные работают. Это не торговая рекомендация и не наша Fibonacci-стратегия.

Из корня проекта запустите:

```bash
backtrader_lab/.venv/bin/python backtrader_lab/run_backtest.py --from-date 2026-04-01 --to-date 2026-04-30
```

Менять параметры можно так:

```bash
backtrader_lab/.venv/bin/python backtrader_lab/run_backtest.py --from-date 2026-04-01 --to-date 2026-06-30 --fast 20 --slow 50 --cash 10000 --commission 0.0006
```

Результат показывает стартовый и финальный капитал, PnL, число закрытых сделок, win rate и максимальную просадку.

## Базовая стратегия Fibonacci

Перенесена первая версия нашей 15-минутной стратегии: подтверждённый импульс, лимитный вход от 0.5 Fibonacci, стоп за началом импульса и тейк на расширении −0.18. Без безубытка и частичной фиксации.

```bash
backtrader_lab/.venv/bin/python backtrader_lab/run_orderflow_fibonacci.py --from-date 2026-04-01 --to-date 2026-04-30
```

Backtrader выполняет расчёт локально. Чтобы поменять торговые правила, нужно изменить или добавить отдельный класс стратегии; для повторных запусков кода менять не нужно.
